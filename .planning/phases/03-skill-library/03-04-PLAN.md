| T-03-04-01 | Tampering | Argument injection via `diff_base_ref` (e.g., `diff_base_ref: "--exec=rm -rf /"`) | mitigate | `git diff` does not have an `--exec` option, but pathological refs that start with `-` could be interpreted as flags.
